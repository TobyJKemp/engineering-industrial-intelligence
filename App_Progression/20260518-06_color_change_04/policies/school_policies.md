# School Color Policies

This markdown file is the guardrails source for allowed school brand colors.

Only the policies listed below are valid.

```yaml
policies:
  - id: michigan
    school: University of Michigan
    allowed_colors:
      - name: Michigan Blue
        value: "#00274C"
      - name: Michigan Maize
        value: "#FFCB05"
      - name: White
        value: "#FFFFFF"
  - id: stanford
    school: Stanford University
    allowed_colors:
      - name: Stanford Cardinal
        value: "#8C1515"
      - name: Palo Alto Green
        value: "#175E54"
      - name: Cool Gray
        value: "#4D4F53"
```
