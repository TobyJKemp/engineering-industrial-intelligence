# Temporary Organizations

## One-Sentence Summary
Temporary Organizations form for specific purpose and fixed duration—achieving objective then disbanding—enabling focused effort without permanent overhead.

## The Core Idea
- **Time-bound:** Fixed start and end date
- **Purpose-driven:** Clear objective
- **Focused:** Dedicated to single purpose
- **Dissolution:** Deliberately disbands when done

## How It Works
1. **Define purpose and timeline:** What's objective? When's deadline?
2. **Assemble team:** Bring together needed people
3. **Execute:** Focused work on objective
4. **Dissolve:** Disband when objective achieved

## Think of It Like This
Temporary organizations are like **committee for specific purpose**: Formed to address problem; works for defined period; dissolves when done.

## The "So What?" Factor
- **Focus:** Dedicated attention to objective
- **Efficiency:** No permanent overhead
- **Urgency:** Time pressure motivates
- **Discontinuity:** Learning doesn't persist

## Watch Out For
⚠️ **Knowledge loss:** Learning disperses with team.
⚠️ **Lack of continuity:** Can't build on previous work.
⚠️ **Member disengagement:** No permanence can reduce commitment.
⚠️ **Inefficiency from startup:** Each new temporary org requires startup effort.

## Practical Checklist

Before creating temporary organization:
- [ ] Is purpose clear and time-bounded?
- [ ] Can you assemble necessary team?
- [ ] Are timeline and deadline specific?
- [ ] Is success criteria measurable?
- [ ] Can you capture learning as work happens?
- [ ] Are resources allocated for full duration?
- [ ] Is governance clear for temporary org?
- [ ] Can temporary org integrate with permanent organization?
- [ ] Is there mechanism to transition results to permanent org?
- [ ] Can you disband cleanly when complete?
