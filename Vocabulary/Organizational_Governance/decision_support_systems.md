## One-Sentence Summary
Decision Support Systems provide information, analysis, and recommendations to decision-makers—improving decision quality and speed through data-driven insights.

## The Core Idea
- **Information provision:** Systems provide relevant information
- **Analysis:** Analyze information
- **Visualization:** Present data clearly
- **Recommendations:** Suggest actions
- **Human authority:** Humans retain decision authority

## How It Works
1. **Design support:** What information do decision-makers need?
2. **Build system:** Create system providing information
3. **Integrate data:** Connect to relevant data sources
4. **Visualize:** Present data clearly
5. **Support:** Help decision-makers use system

## Think of It Like This
Decision support systems are like **pilot dashboard**: Presents relevant flight information; helps pilot make decisions; pilot retains control.

## The "So What?" Factor
- **Better decisions:** Information-driven decisions
- **Speed:** Faster decision-making
- **Consistency:** Consistent analysis
- **Learning:** Can learn from system outputs

## Watch Out For
⚠️ **Over-reliance:** Decision-makers become dependent.
⚠️ **Bad data:** System only as good as data.
